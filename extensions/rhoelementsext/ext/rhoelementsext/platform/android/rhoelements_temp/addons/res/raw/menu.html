Running RhoElements Browser for the first time.

  • Welcome
  • Config.xml
Welcome to RhoElements Browser. This is the default startpage. This page has been provided to get you and your application up and running with RhoElements Browser. There is just one step required for first use:
  1. Change the default Config.xml
    available at "/Android/data/Installed_Dir_Location"
    location. Here "Installed_Dir_Location"
    is the path where RhoElements is
    installed in the device
The Config.xml sets default settings for all your RhoElements Browser applications. To run your application:
  1. Open for editing:

    Config.xml

  2. Set the "value" parameter of the "StartPage" tag to the URL of your application's first page.
  3. Save Config.xml, and restart RhoElements Browser.
